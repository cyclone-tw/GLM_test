# 3D 動畫抽籤系統設計文件

## 概述

一個可自訂名單和獎項的 3D 動畫抽籤 HTML 應用程式，採用球體式 3D 動畫效果，支援多種主題切換和豐富的互動功能。

## 核心功能

### 1. 3D 動畫效果

**球體式抽籤動畫：**
- 名字以 3D 文字或卡片形式漂浮在球體空間中
- 點擊抽獎時，球體加速旋轉
- 旋轉逐漸減速後，中獎者名字飛出並放大顯示
- 使用 Three.js 實現 3D 渲染

**中獎動畫：**
- 煙火特效
- 彩帶/紙屑飄落
- 中獎者名字發光放大

### 2. 獎項與名單管理

**獎項處理邏輯：**
- 獎項與名單一起抽：先設好獎項清單，每次抽獎從名單抽出得獎者並綁定獎項
- 中獎者從名單移除，不重複中獎
- 獎項抽完後可繼續抽剩餘名單（無獎項綁定）

**輸入方式：**
- 列表管理介面：新增/刪除/編輯/拖曳排序
- 批次匯入：文字框貼上，一行一項
- Google Sheets 匯入：透過 Google Sheets API（使用者提供 API Key）
- Google 表單匯入：讀取表單回應資料

### 3. 主題系統

**預設主題：**
1. 科技感 - 深色背景、霓虹光效、粒子特效
2. 節慶派對 - 彩色繽紛、金幣彩帶、熱鬧歡樂
3. 簡約現代 - 淺色背景、扁平設計、專業感
4. 自訂 - 使用者可調整顏色、背景等

**主題切換：**
- 即時切換，不需重新載入
- 主題設定儲存於 localStorage

### 4. 音效系統

- 滾動音效：抽獎進行時播放
- 中獎音效：揭曉中獎者時播放
- 音效開關：使用者可關閉音效

### 5. 全螢幕模式

- 投影模式：隱藏設定區，只顯示 3D 動畫區
- 適合會議/活動現場投影使用
- 快速鍵切換（如 F 鍵）

### 6. 歷史記錄

- 記錄每次抽獎結果（時間、獎項、中獎者）
- 可查看歷史記錄列表
- 可清除歷史記錄
- 儲存於 localStorage

### 7. 匯出功能

**結果匯出格式：**
- CSV：試算表格式
- JSON：結構化資料
- 圖片：中獎畫面截圖（使用 html2canvas）

## 技術架構

### 前端技術

| 技術 | 用途 |
|------|------|
| HTML5 | 結構 |
| CSS3 | 樣式與動畫 |
| JavaScript | 邏輯控制 |
| Three.js | 3D 渲染 |
| html2canvas | 截圖匯出 |

### 檔案結構

```
lottery.html          # 單一 HTML 檔案（內含 CSS 和 JS）
```

### 資料儲存

- 名單、獎項：localStorage
- 歷史記錄：localStorage
- 主題設定：localStorage
- Google API Key：localStorage（使用者自行輸入）

### 外部 API

**Google Sheets API：**
- 使用者需自行申請 API Key
- 讀取公開或授權的試算表
- 轉換為名單格式

## 使用者流程

```
1. 開啟頁面
2. 選擇主題
3. 輸入/匯入名單
4. 輸入/匯入獎項
5. 點擊「開始抽獎」
6. 觀看 3D 動畫
7. 中獎者揭曉（含動畫與音效）
8. 重複步驟 5-7 直到完成
9. 匯出結果（選用）
```

## 錯誤處理

- 名單為空時，提示「請先加入名單」
- 獎項為空且抽獎時，提示「無獎項，將只抽出名字」
- Google API Key 無效時，顯示錯誤訊息
- 網路錯誤時，提示「無法連線，請檢查網路」

## 效能考量

- Three.js 使用 WebGL 渲染
- 名單超過 100 人時，只顯示部分名字在球體上
- 動畫使用 requestAnimationFrame
- 響應式設計，支援不同螢幕尺寸
